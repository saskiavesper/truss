# Truss

Truss is a B2B project management and orchestration platform designed for an AI interactive workspace. It bridges the
gaps between fragmented customer systems (CRMs), delivery lifecycles, financial data (ERPs), and unstructured human 
thoughts.

Traditional project management software introduces administrative friction—such as clicking through nested menus,
manual state tracking, and scattered documentation—which engineering and delivery teams dislike. Truss removes this 
friction by transforming bookkeeping into an automated, real-time network of events.

## Getting Started

Requirements:
 - [mise](https://mise.jdx.dev/)

Clone the repository, cd into it then run:
 ```
 mise trust
 mise setup
 cargo test-all
 ```
